# Demo Video Script — 3-5 Minutes

## Title: "AI Talent Scouting Agent — Automated Candidate Matching in Under 2 Seconds"

---

## SCENE 1: Introduction (0:00 - 0:30)

**Narration:**
"Meet the AI Talent Scouting Agent — a production-grade system that automates the most time-consuming part of hiring: finding qualified candidates.

Instead of manually searching through hundreds of resumes, this system takes just ONE job description and intelligently ranks the best candidates in under 2 seconds.

Let me show you how it works."

**Visual:**
- Show system diagram for 2-3 seconds
- Then show main dashboard UI

---

## SCENE 2: Input Job Description (0:30 - 1:15)

**Action:** 
- Click in the job description text area
- Paste this realistic job description:

```
Senior Backend Engineer - TechCorp

We are hiring a Senior Backend Engineer with strong experience in Python and FastAPI.

Requirements:
- 4+ years of experience in backend development
- Strong proficiency in Python
- Experience with FastAPI or Django
- Knowledge of databases like PostgreSQL or MongoDB
- Experience with cloud platforms such as AWS or GCP
- Familiarity with Docker and Kubernetes is a plus
- Location: Remote / Bangalore

We're looking for someone who can lead technical decisions and mentor junior developers.
```

**Narration:**
"First, we paste a job description. This could be from LinkedIn, your internal job posting, or anywhere else. The system accepts raw, unstructured text — no special formatting needed.

Notice the flexibility: it mentions specific technologies, experience levels, and preferences — all things recruiters care about."

**Visual:**
- Text appears in input box
- Highlight key parts being recognized (4+, Python, FastAPI, PostgreSQL)

---

## SCENE 3: Submit & Processing (1:15 - 1:45)

**Action:**
- Click "Find Talent" button
- Show loading state for 2-3 seconds

**Narration:**
"When we click 'Find Talent', the system immediately:

1. **Parses the job description** — extracts the title (Senior Backend Engineer), required experience (4 years), skills (Python, FastAPI, PostgreSQL), and domain (Backend)

2. **Loads candidates** — retrieves candidate profiles from the database

3. **Scores each candidate** using a sophisticated 4-factor algorithm

4. **Simulates outreach** — generates realistic recruiter-candidate conversations

5. **Ranks results** — displays the best matches at the top

All of this happens in just 1-2 seconds."

**Visual:**
- Show system processing (animated steps)
- Show timing indicator: "Processing: 1.2 seconds"

## Production Notes

### Timing
- Full script: 5-5:30 minutes (adjust content in Scene 6-7 if needed to fit)
- Can be compressed to 3 minutes by skipping Scene 8 & 9

### Visual Elements Needed
1. System diagram (from ARCHITECTURE.md)
2. Screenshot of main dashboard
3. Job description input field with example text
4. Loading animation (1-2 sec)
5. Candidate cards with rankings
6. Scoring breakdown (4-factor grid)
7. Strengths/gaps lists
8. Conversation simulation
9. Comparison view of Rank 1, 2, 3
10. Optional: Code snippets and system architecture

### Tools for Screen Recording
- **macOS**: QuickTime (built-in)
- **Windows**: OBS Studio (free), Camtasia (paid)
- **Web**: Loom (cloud-based, easy sharing)

### Audio
- Use clear, professional narration (AI voice or human)
- Add subtle background music during processing/loading scenes
- Keep conversational tone — avoid too technical for general audience

### Editing
- Keep transitions smooth but minimal
- Use color highlighting to draw attention to key metrics
- Slow down at critical moments (e.g., revealing scores)
- Speed up non-critical transitions

## Alternative: Live Demo Script (Without Pre-recording)

If presenting live instead:

1. **Start with dashboard** (0:00)
2. **Paste job description** - Invite audience member to suggest one (0:30)
3. **Click "Find Talent"** (1:00)
4. **Explain scoring while loading** (1:15-2:00)
5. **Show results** (2:00)
6. **Live Q&A** - "What would you like to know about Rank #1?" (2:30-5:00)
7. **Wrap up** with next steps (5:00)

This approach is more engaging and allows for real-time customization based on audience interests.
